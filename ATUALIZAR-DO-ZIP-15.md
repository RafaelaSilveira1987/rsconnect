# Atualizar do ZIP 15 para o ZIP 16

1. Substitua os arquivos do projeto pelos arquivos deste ZIP.
2. Faça commit/push para o GitHub.
3. Faça redeploy do serviço `rsconnect` no EasyPanel.
4. Não rode migration.
5. Acesse a aplicação e pressione `Ctrl + F5`.

## Validação recomendada

- Abrir `/conversations`.
- Selecionar uma conversa.
- Clicar em "Dados do lead".
- Confirmar que a gaveta lateral abre completa, com formulário visível e botão "Salvar alterações".
- Confirmar que o painel não ocupa espaço fixo no chat quando fechado.
