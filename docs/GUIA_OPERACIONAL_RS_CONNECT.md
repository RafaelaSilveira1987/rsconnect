# Guia operacional RS Connect

## Rotina diária

- Abrir Monitoramento.
- Conferir conversas novas e mensagens não lidas.
- Conferir se o backup automático registrou sucesso.
- Verificar alertas de cobrança e webhooks.

## Rotina semanal

- Revisar checklist de implantação das empresas.
- Testar envio/recebimento de uma instância ativa.
- Validar se os agentes de IA seguem o prompt e regra de horário.
- Conferir faturas e régua de cobrança.

## Implantação de novo cliente

1. Criar empresa.
2. Criar usuário administrador do cliente.
3. Conectar instância WhatsApp.
4. Criar agente IA.
5. Configurar pré-agendamento, se aplicável.
6. Configurar termos LGPD.
7. Executar onboarding e teste final.
8. Marcar checklist como pronto para teste/operação.
