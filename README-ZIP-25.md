# RS Connect — ZIP 25 — Central de Ajuda e Beta Comercial

Este pacote adiciona uma central de ajuda dentro do sistema e um painel de prontidão para beta comercial.

## Inclui

- Rotas `/ajuda`, `/central-ajuda` e `/docs`.
- Rotas `/beta-comercial` e `/versao-beta` para Super Admin.
- Central de ajuda para RS Admin e clientes.
- Painel de critérios de prontidão comercial.
- Release notes internas.
- Rotina operacional diária/semanal.
- Documentos em `docs/`.
- Escopo alinhado: sem campanhas e sem disparos em massa.

## Migration

Não há migration obrigatória neste ZIP.

## Depois de aplicar

1. Subir os arquivos.
2. Fazer redeploy.
3. Pressionar Ctrl+F5.
4. Acessar `/beta-comercial` como Super Admin.
5. Acessar `/ajuda` como Super Admin ou cliente.
